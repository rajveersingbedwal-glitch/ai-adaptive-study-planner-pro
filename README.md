# AI Adaptive Study Planner Pro

AI Adaptive Study Planner Pro is a smart web-based study planning system that dynamically generates personalized study schedules based on subject exam dates, weak subject prioritization, and available daily study hours.

## Features

- Multi-subject scheduling
- Intelligent exam-based distribution
- Weak subject prioritization
- Progress tracking
- Performance trend visualization
- Study-break timer
- Local data persistence

## Built With

- HTML
- CSS
- JavaScript
- LocalStorage API
